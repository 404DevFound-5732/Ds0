# 线性表

##  一句话理解
 一排数据，可以用数组或链表实现


##  基本概念
- 线性结构
- 一对一关系
## 特点
- 存在唯一的一个被称作“第一个”的数据元素
- 存在唯一的一个被称作“最后一个”的数据元素
- 除第一个元素外，结构中每个数据元素均只有一个前驱
- 除最后一个元素外，结构中每个数据元素均只有一个后继


---

##  顺序表
### 概念
逻辑上相邻的元素，实际物理存储空间也是连续的
###  解决什么问题
- 用连续内存存储数据
- 支持快速访问

###  结构（核心）
- 定义顺序表
```c
#define MAXSIZE 100
typedef struct{
    //ElemType data[MAXSIZE];// 数据区
    ElemType *data;
    int length;             // 当前长度
} SeqList;
```

- 顺序表初始化
```c
SeqList* initList()
{
    SeqList *L = (SeqList*)malloc(sizeof(SeqList));
    L->data = (ElemType*)malloc(sizeof(ElemType) * MAXSIZE)
	L->length = 0;
    return L;
}
```

- 尾部添加元素
```c
int appendElem(SeqList *L, ElemType e)
{
	if (L->length>=MAXSIZE)
	{
		printf("顺序表已满\n");
		return 0;
	}

	L->data[L->length] = e;
	L->length++;
	return 1;
}

```
###  操作
- 查找：下标访问 O(1)

- 插入：移动元素 O(n)

```text
原：
[1][2][3][4][]

→ 插入5到位置1,元素从最后一个开始向后移动
过程：
[1][][2][3][4]
结果：
[1][5][2][3][4]
```

```c

```
><mark>插入数据的最好时间复杂度为O(1)，最坏为O(n)</mark>

- 删除：覆盖元素 O(n)
```text
原：
[1][2][3][4]
过程:
[1][3][4][4]//相当于一个个覆盖，但是最后length长度减一
结果:
[1][3][4]
```


```c
int deleteElem(SeqList *L, int pos, ElemType *e)
{
	*e = L->data[pos-1];//删除的元素
	if (pos < L->length)
	{
		for (int i = pos; i < L->length; i++)//从删除元素的位置循环
		{
			L->data[i-1] = L->data[i];
		}	
	}
	L->length--;
	return 1;
}
```
- 反转链表
```c
1->2->3->4->5->6->NULL
1.利用三个指针，first指针指向NULL，second指针指向头指针的next,建一个NULL<-1->2……->NULL
2.三指针遍历链表改变‘->’方向，直到second指向NULL
//反转链表
Node* reverseList(Node* L)
{
	Node* first = NULL;
	Node* second = L->next;
	Node* third
	while(second != NULL)
	{
		third = second->next;
		second->next = first;
		first = second;
		second = third;
	}
}
```

###  特点
- 查询快
- 插入删除慢

###  易错点
- 下标越界
- 插入时顺序写错
- 没判断是否满

##  链表

###  解决什么问题
- 顺序表插入/删除需要移动元素（效率低）
- 链表通过“指针连接”避免移动

---

###  结构（核心）
head
 ↓
[ data | next ] → [ data | next ] → [ data | next ] → NULL
👉 组成：
- data：数据
- next：指向下一个节点  

👉 特点：
- 节点在内存中不连续
- 靠指针连接
```c
typedef int ElemType;

typedef struct node{
	ElemType data;
	struct node *next;
}Node;
```
---

- 初始化


- 删除
```text

```

```c
int deletNode(Node* L,int pos){
```

- 释放链表
```c
//指针p指向头节点的第一关节点2.判断指针p是否指向空节点3如果p不为空,用q记录指针p的后继节点3.释放指针p指向的节点4.指针p和指针q指向同一个节点，循环上面的操作
void freeList(Node *L)
{
	Node *p = L
	Node *q;
	while (p != NULL)
	{
		q = p->next;
		free(p);
		p = q;
	}
	L->next = NULL;
}
```

### 单链表应用
- 快慢指针
```c
//查找倒数第k个节点
1.快指针比慢指针多移动k步后两者同时移动
2.当快指针指向空置时，输出慢指针
int findNodeFS(Node *L, int k)
{
	Node* fast = L->next;
	Node* slow = L->next;
	for (int i = 0; i < k; i++)
	{
		fast = fast->next;
	}
	while(fast != NULL)
	{
		fast = fast->next;
		slow = fast->next;
	}
	printf("倒数第%d个节点为：%d",k, slow->data);
}
```
- 删除绝对值相同的节点
```c
1.建立数组储值
2.遍历数组，初始值为0
3.获取绝对值，若无则数组值＋1，有则删除节点
//数组存储的值n的值不能超过链表中最大值，若链表最大数字为n，则数组从0开始到21有22个数字，即n+1个空间
int removeNode(Node* L int n)
{
	Node* p = L;
	int index;//数组下标
	int *q = (int*)malloc(sizeof(int*)*(n+1))
	
	//遍历数组，初始化为0
	for (int i = 0 ; i < n+1; i++)
	{
		*(q+i) = 0;
	}
while(p->next != NULL)
	{
		//abs获取绝对值
		index = abs(p->next->data);
		if(*(q+index) == 0)
		{
			*(q + index) = 1;
			p = p->next;
		}
		else
		{
			Node *temp = p->next;
			p->next = temp->next;
			free(temp);
		}
		
	}
	free(q);
}
```

### 🎯 特点（必背）
- 插入删除快（O(1)）
- 查找慢（O(n)）
- 不需要连续内存

---

### ⚠️ 易错点（非常重要！！！）
- 指针顺序写反（会断链）
- 忘记保存 next
- 空指针访问（NULL）
- head 处理错误

---

### 🔁 一句话总结
👉 链表 = 节点 + 指针连接（不连续内存）
## 🔥 顺序表 vs 链表

| 对比 | 顺序表 | 链表 |
|------|--------|------|
| 存储 | 连续 | 不连续 |
| 查询 | 快 | 慢 |
| 插入删除 | 慢 | 快 |
| 本质 | 数组 | 指针 |
