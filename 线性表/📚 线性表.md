# 线性表

##  一句话理解
 一排数据，可以用数组或链表实现


##  基本概念
- 线性结构
- 一对一关系
## 特点
- 存在唯一的一个被称作“第一个”的数据元素
- 存在唯一的一个被称作“最后一个”的数据元素
- 除第一个元素外，结构中每个数据元素均只有一个前驱
- 除最后一个元素外，结构中每个数据元素均只有一个后继


---

##  顺序表
### 概念
逻辑上相邻的元素，实际物理存储空间也是连续的
###  解决什么问题
- 用连续内存存储数据
- 支持快速访问

###  结构（核心）
- 定义顺序表
```c
#define MAXSIZE 100
typedef struct{
    //ElemType data[MAXSIZE];// 数据区
    ElemType *data;
    int length;             // 当前长度
} SeqList;
```

- 顺序表初始化
```c
SeqList* initList()
{
    SeqList *L = (SeqList*)malloc(sizeof(SeqList));
    L->data = (ElemType*)malloc(sizeof(ElemType) * MAXSIZE)
	L->length = 0;
    return L;
}
```

- 尾部添加元素
```c
int appendElem(SeqList *L, ElemType e)
{
	if (L->length>=MAXSIZE)
	{
		printf("顺序表已满\n");
		return 0;
	}

	L->data[L->length] = e;
	L->length++;
	return 1;

```
###  操作
- 查找：下标访问 O(1)

- 插入：移动元素 O(n)

```text
原：
[1][2][3][4][]

→ 插入5到位置1,元素从最后一个开始向后移动
过程：
[1][][2][3][4]
结果：
[1][5][2][3][4]
```


><mark>插入数据的最好时间复杂度为O(1)，最坏为O(n)</mark>

- 删除：覆盖元素 O(n)
```text
原：
[1][2][3][4]
过程:
[1][3][4][4]//相当于一个个覆盖，但是最后length长度减一
结果:
[1][3][4]
```


```c
int deleteElem(SeqList *L, int pos, ElemType *e)
{
	*e = L->data[pos-1];//删除的元素

	{
		for (int i = pos; i < L->length; i++)//从删除元素的位置循环

		}	
	}

	return 1;
}
```

- 反转链表
```c
1->2->3->4->5->6->NULL
1.利用三个指针，first指针指向NULL，second指针指向头指针的next,建一个NULL<-1->2……->NULL
2.三指针遍历链表改变‘->’方向，直到second指向NULL
//反转链表
Node* reverseList(Node* L)
{
	Node* first = NULL;
	Node* second = L->next;
	Node* third
	while(second != NULL)
	{
		third = second->next;
		second->next = first;
		first = second;
		second = third;
	}
}
```

###  特点
- 查询快
- 插入删除慢

###  易错点
- 下标越界
- 插入时顺序写错
- 没判断是否满

##  链表

###  解决什么问题
- 顺序表插入/删除需要移动元素（效率低）
- 链表通过“指针连接”避免移动

---

###  结构（核心）
head
 ↓
[ data | next ] → [ data | next ] → [ data | next ] → NULL
👉 组成：
- data：数据
- next：指向下一个节点  

👉 特点：
- 节点在内存中不连续
- 靠指针连接
```c
typedef int ElemType;

typedef struct node{
	ElemType data;
	struct node *next;
}Node;
```

```c
int removeNode(Node* L int n)

	int index;//数组下标

		*(q+i) = 0;
```

| 查询 | 快 | 慢 |
| 插入删除 | 慢 | 快 |

# 栈和队列

##  栈（Stack）

###  定义
- 一种“先进后出”（LIFO）的线性结构
- 类似：叠盘子

---

###  特点
- 只能在“栈顶”操作
- 插入：入栈（push）
- 删除：出栈（pop）

---

###  示意图
      栈顶
       ↓
     ┌───┐
     │ 3 │
     ├───┤
     │ 2 │
     ├───┤
     │ 1 │
     └───┘ -> 栈底
 入栈顺序：1 → 2 → 3  
 出栈顺序：3 → 2 → 1  

---

### ⚙️ 基本操作

#### 栈的顺序结构实现（动态内存分配）
```c
#include <stdio.h>
#include <stdlib.h>
#define MAXSIZE 100
typedef int ElemType;

typedef struct 
{
	ElemType *data;
	int top;
	
}Stack;

//初始化
Stack* initStack()
{
	Stack* s = (Stack*)malloc(sizeof(Stack));
	s->data = (ElemType*)malloc(sizeof(ElemType) * MAXSIZE);
	s->top = -1;
	return s;
}
//判断是否为空栈
int isEmpty(Stack *s)
{
	if (s->top == -1)
	{
		printf("空的");
		return 1;
	}
	else
	{
		return 0;
	}
}
//进栈
int push(Stack *s, ElemType e)
{
	if (s->top >= MAXSIZE-1)
	{
		print("满了");
		return 0;
	}
	s->top++;
	s->data[s->top] = e;
	return 1;
}
//出栈
int pop(Stack *s, ElemType *e)
{
	if (s->top == -1)
	{
		printf("空的");
		return 0;
	}
	*e = s->data[s->top];
	s->top--;
	return 1;
}
//获取栈顶元素
int getTop(Stack *s, ElemType *e)
{
	if (s->top == -1)
	{
		printf("空的\n");
		return 0;
	}
	*e = s->data[s->top];
	return 1;
}
int main(int argc, char const *argv[])
{
	
	Stack *s = initStack();
	push(s, 10);
	push(s, 20);
	push(s, 30);
	
	ElemType e;
	pop(s, &e);
	printf("%d\n", e);

	getTop(s, &e);
	printf("%d\n", e);
	
	return 0;
}
```


#### 栈的链式结构实现

```c
	ElemType data;
	struct stack *next;
}Stack;

//初始化
Stack* initStack()
{
	Stack *s = (Stack*)malloc(sizeof(Stack));
	s->data = 0;
	s->next = NULL;
	return s;
}

//判断栈是否为空
int isEmpty(Stack *s)
{
	if (s->next == NULL)
	{
		printf("空的\n");
		return 1;
	}
	else
	{
		return 0;
	}
}

//进栈/压栈
int push(Stack *s, ElemType e)
{
	Stack *p = (Stack*)malloc(sizeof(Stack));
	p->data = e;
	p->next = s->next;
	s->next = p;
	return 1;
}

//出栈
int pop(Stack *s, ElemType *e)
{
	if(s->next == NULL)
	{
		printf("空的\n");
		return 0;
	}
	*e = s->next->data;
	Stack *q = s->next;
	s->next = q->next;
	free(q);
	return 1;
}

//获取栈顶元素
int getTop(Stack *s, ElemType *e)
{
	if (s->next == NULL)
	{
		printf("空的\n");
		return 0;
	}
	*e = s->next->data;
	return 1;
}

int main(int argc, char const *argv[])
{
	
	Stack *s = initStack();
	push(s, 10);
	push(s, 20);
	push(s, 30);
	
	ElemType e;
	pop(s, &e);
	printf("%d\n", e);

	getTop(s, &e);
	printf("%d\n", e);
	
	return 0;
}
```

### 🎯 应用（很重要）

- 括号匹配
- 表达式计算
- DFS（深度优先搜索）

---

### ⚠️ 易错点
- 栈空还 pop（越界）
- 栈满还 push（溢出）

---

### 🔁 一句话总结
👉 栈 = 先进后出，只操作栈顶
## 📖 队列（Queue）

### 📌 定义
- 一种“先进先出”（FIFO）的线性结构
- 类似：排队买饭

---

### 🧠 特点
- 一端进（队尾）
- 一端出（队头）

---

### 📊 示意图
队头 → [ 1 ] [ 2 ] [ 3 ] ← 队尾
👉 入队顺序：1 → 2 → 3  
👉 出队顺序：1 → 2 → 3  

---

### ⚙️ 基本操作

## ⚔️ 栈 vs 队列

| 对比 | 栈 | 队列 |
|------|----|------|
| 规则 | LIFO | FIFO |
| 操作位置 | 只在一端 | 两端 |
| 插入 | push | enqueue |
| 删除 | pop | dequeue |
| 应用 | 递归/DFS | BFS |

---

👉 核心区别：
- 栈：后进先出
- 队列：先进先出
